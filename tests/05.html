Records - List Field Value in Table

  1. Open Records, create a new database named '05'.
  2. Select Insert- Table from the menu.
  3. In the new table wizard, create a name field as the default character type.
  4. In the next field, click on the field type menu and select list.
  5. A dialog will pop up allowing the user to add values that can be selected from.
  6. Add the list 'banana', 'orange', 'apple' and 'pear'.
  7. Click OK and create the table.
  8. Create a few rows in table view, ensure the drop down list has the fruits in.
  9. Create another couple of rows in form view, ensuring the drop down list works correctly, listing the fruits.
  10. Ensure the data is written correctly by using a text editor to read the row data.
  11. Save the database to tests.
