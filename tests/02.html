Records File Test - Reading File without Records

  1. Fetch the Records music sample database onto disk.
  2. The file should have a .sql ending, right click on it.
  3. Select a text editor, such as Notepad++, Notepad, or gedit/Kate/Mousepad. Open with one of these programs.
  4. See if you can read UPPERCASE and lowercase text, with names of music albums.
  5. Take a screenshot of the opened file. Save in this tests folder, as '2winxp.jpg', '2win7.jpg' or '2linux.jpg' depending on OS.
