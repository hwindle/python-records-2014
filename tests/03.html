Custom Data Fields, ready made

  1. Open Records, create a new database named '03'.
  2. Select Insert- Table from the menu.
  3. In the new table wizard, create a name field as the default character type.
  4. In the next field, click on the field type menu and select validating.
  5. A dialog will pop up showing common data types, such as credit card numbers, and postcodes.
  6. Select the National Insurance number type from the list, and click 'Add'
  7. Type in the field name 'NI_num' for that field.
  8. Make a similar field called 'phone' and select a UK phone number data type.
  9. Create the table and start entering a few rows of data.
  10. Try to type text or an empty phone number into either the NI_num or phone fields.
  11. Typing the wrong data should bring up a format warning box, containing the format rules.
  12. Save the database to tests.
