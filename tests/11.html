Records - Queries

  1. Open Records, with the numerical database.
  2. Go to query view.
  3. Make sure the GUI looks the same as on other OS'es.
  4. Query the data using text, number values and limiting the number of rows. Save the queries.
  5. See how easy it is to run the query again.
  6. Save the database to tests.
  7. Reopen the same database, and run the saved query views again.
