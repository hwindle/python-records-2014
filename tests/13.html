Records - Internet Lookup

  1. Open Records, open the wishlist database.
  2. Go to table view. Pick a row.
  3. In the 'store' column, select the text.
  4. Right click on the store name.
  5. Select 'Internet lookup' from below the 'cut', copy paste menu.
  6. Verify that it opens a browser with google, searching for the store name.
