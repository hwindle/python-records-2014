Records - Table sheet view

  1. Open Records, load the database named '06' containing a wishlist.
  2. Switch to table sheet view in the 'View' menu.
  3. Ensure the 50-100 rows of data are displayed correctly.
  4. Sort by store by clicking on the header bar named 'Store'.
  5. Sort by Item (the default).
  6. Click once on the Price header to sort descending, then again to sort by ascending.
  7. Save the database via save as '06-sort', see if 06 and 06-sort differ, if they are then the test has failed.
