Custom Data Fields, - Creation (beta feat.)

  1. Open Records, create a new database named '04'.
  2. Select Insert- Table from the menu.
  3. In the new table wizard, create a title field as the default character type.
  4. In the next field, click on the field type menu and select validating.
  5. A dialog will pop up showing common data types, select the new button.
  6. Create a data type called 'part_num' with two numbers at the start, and 3 uppercase letters after that.
  7. Type in the field name 'part' for that field. Choose the data type you just created.
  8. Create the table and start entering : yoyo 01ABC, umbrella 99YOU
  9. Try to type text or a wrong part number into the 'part' field.
  10. Typing the wrong data should bring up a format warning box, containing the format rules that you defined.
  11. Save the database to tests.
