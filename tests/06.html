Records - Test Form View

  1. Open Records, create a new database named '06'.
  2. Create a new table from the menus, with a name of 'wishlist'.
  3. Enter the following fields:
    1. Item - variable character, 100 characters max.
    2. Price - numeric field, no maximum.
    3. Quantity - whole number field, maximum of 40.
    4. Brought - Yes/No field
    5. Description - long text field
    6. Category - (list with: furniture, books, films, games, electronics, textiles, ornaments)
    7. Store - URL validated field
  4. Create the database and you should be taken to a form view straightaway.
  5. Enter 50 rows of data from a separate file, one by one into the database.
  6. a) Use the add row button to add a new row (an empty form should appear with the row number incremented).
  7. b) Go back one row, then delete a row using the delete button, it should disappear and the row after that should appear.
  8. Note down each mistake in the user interface.
  9. Save the database.
